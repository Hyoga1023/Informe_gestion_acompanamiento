# Informe_gestion_acompanamiento
Sistema de Actas de Visitas - Protección S.A.
Aplicación web para registrar y gestionar visitas a grandes empleadores por parte de asesores de Protección S.A. Permite:

Captura de información: Datos básicos de la visita, asistentes, lista de verificación (deudas e inconsistencias con valores en pesos), canales digitales, actualización de datos, temas generales, agendamiento y encuesta de satisfacción.
Gestión dinámica: Checkboxes que despliegan campos de observación, validaciones inteligentes (si hay observación, el valor es obligatorio), formato automático de moneda COP, y campos condicionales según respuestas.
Exportación: Botones para guardar localmente (localForage), generar Excel (estructura fija para análisis posterior) y PDF (para firmas físicas).
Stack técnico: HTML5, CSS3, JavaScript vanilla, sin backend (todo del lado del cliente). Diseño corporativo con los colores de Protección (amarillo #dfca0e y azul #091057).
Estado actual: Esqueleto completo y funcional, pendiente de implementar la lógica de guardado y exportación tras validación con usuarios.
